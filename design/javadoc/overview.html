This design documentation describes the packages of the Node Grader software system. The system is comprised of two application programs:
  • The Node Grader web interface, for regular users to manage and view their calendars.
  • the Node Grader Server, which runs in the background on central hosts to control communication among users' local computers and the host.
See the companion requirements specification document has complete details of grader tool functionality.
