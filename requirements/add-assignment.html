2.4.1. Adding a New Assignment

To create a new assignment, the user chooses the 'Assignment' menu command. The user chooses 'New Assignment' from the list of items to add a new assignment; this takes the form of the following, Figure 1.

Figure 1: Add Assignment path.


Figure 2 shows the response to the users selection of the 'Assignment New Assignment' command. The figure shows a dialog box for adding a new assignment. The dialog box only takes in a name in a text box, and an input box for the categories it goes into. The user confirms their choices with the 'Add Assignment' button below. Confirming the assignment creates a new assignment in the spreadsheet.

Figure 2: Add Assignment dialog.


When the user selects the 'Add Assignment' button, the system inputs the newly created Assignment in Figure 3.

Figure 3: Assignment Added view.


To create another assignment, the user chooses the 'Assignment' menu command again. The user chooses 'New Assignment' from the list of items to add a new assignment; takes the form of the following, Figure 4.

Figure 4: Add Assignment path.

The dialog box only takes in a name in a text box, and an input box for the categories it goes into. The user confirms their choices with the 'Add Assignment' button below. Confirming the assignment creates a new assignment in the spreadsheet.

Figure 5: Add Assignment dialog.


When the user selects the 'Add Assignment' button, the system inputs the newly created Assignment in Figure 6.
