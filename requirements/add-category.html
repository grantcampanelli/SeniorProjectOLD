2.2.2. Adding a New Category

To create a new category, the user chooses the 'Assignment' menu command. The user chooses 'New Assignment Category' from the list of items to add a new category; this takes the form of the following, Figure 1.

When the user selects the 'Add Category' button, the system inputs the newly created category as shown in Figure 3.
