2. Functional Requirements

The primary functionality the Grader Tool is for managing classes and inputting student grades. Grader operations are performed by professors on the classes they teach, as well as by students, who may want to view their grades. The Grader Tool also provides functionality for class administration by setting a variety of grade policies and visualization of the class grade curve.

Following an overview of the Grader user interface, details of system use are presented in the following scenarios:

  • an individual user adding new items to a spreadsheet
  • an individual user editing existing items in Grader
  • an individual user using the graphical tools provided
  • an individual user performing administrative actions
These scenarios cover all functional requirements for the system.
