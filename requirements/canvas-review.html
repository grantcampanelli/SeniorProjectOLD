1.5.6. Instructure Canvas

Instructure Canvas is a web-based learning management system (LMS) used in all levels of education, K-12, as well as in higher ed. It is a all-round management tool that supports features including but not limited to: grading, testing, discussions, collaborations, announcements, and instructor feedback.

Good Features:

  • Students can use all features for free with an account
  • Manages multiple classes for instructors and students
  • Sub-class level groups can be formed and managed
  • Manages student submissions of various formats
  • Students can easily access assignments, grades, and feedback
  • Supports a calendar view of all assignments and tests

Bad Features:

  • It relatively slow compared to other systems
  • Instructor or institurion needs to purchase an account
  • It only supports single-level, complex assignments

Missing Features:

  • Spreadsheet style input interface that is enhanced for grading
  • Course grade curve adjustments
  • Student centered what if analysis
  • Integration with Cal Poly's authentication system
  • Graphical representation of class trend and individual grades
