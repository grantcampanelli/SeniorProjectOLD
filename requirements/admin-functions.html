2.5. Administrative Functions

Scenarios in this section illustrate functions available to the main administrator or instructor of a class. The following functions listed below allow the main instructor to publish grades to students, submit grades to school, export student roster, set or change grade scheme, and set or change late policy of the class:

  • publish grades for students to view
  • submit grades to school's registrar and administrators to use
  • export the student roster as a file
  • set or change class's grading scheme
  • set or change class's late policy
These scenarios cover all administrative requirements for the system for now, but more may be added.
