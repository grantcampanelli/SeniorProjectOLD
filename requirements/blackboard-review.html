1.5.2. Blackboard

Blackboard is one of the most widely used grading application for K-12 Education. They have various products that all work in sync.

Good Features:

  • Assignments can be automatically graded
  • Points can be directly added to the grade database from grader
  • Quizzes can be taken on the platform
  • Tests can be automatically generated and unique to reduce academic dishonesty
  • Work offline
  • Manage grades via spreadsheet view
  • Adding comments to grades
  • Grades can be exported to a spreadsheet
  • New user defined columns

Bad Features:

  • Needs native application to have decent style on mobile application.
  • Difficult for teachers to manipulate data in bulk.
  • No style options for pages.

Missing Features:

  • Cannot have multiple files open
  • No sub-assignment level score management
  • No student grade prediction report
  • No graphical curve setting interface
  • No team grade submission
