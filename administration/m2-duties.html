addNewClassView

Mike Ryu

  1. Click on ‘Add New Class’ to bring up the dialog to add a new class to the Grader.
  2. Once the dialog is up, you can do the following:
  1. Click on Cancel to close the dialog.
  2. Enter class code and class number.
  3. Click on OK to attempt submitting a (mock) request to add a new class.
  4. Click on OK button results in a console log of “OK Button Clicked!”
  5. Depending on if the all required information was given, an alert pop-up will show either confirming the successful request or reporting an input error.

http://localhost 

Michael Murphy

  1. Download and install the latest (v0.12.2) Node JS Executable (These directions assume your path environment variable includes the directory with the node js executable)
  2. cd into the project’s implementation directory
  3. enter: node --harmony main.js (Note: the server will listen on port 80. If this is undesirable, change line 12 of app/config.js (use a port larger than 1024 to avoid the need for sudo on the mac))
  4. wait for the “Application ready” message to appear in the console (this takes half a second usually)
  5. go to http://localhost you should see a login screen
  6. Enter username: admin, password: admin to see an alert box showing that you logged in Successfully
  7. Enter the wrong username and password to see an error message. The server is doing password validation.

Beauchamp

Adding a New Assignment

  1. Click on ‘Add New Assignment’ to bring up the dialog to add a new assignment
  2. Enter in values for the Assignment Name, Weight, Total Score and Category fields
  3. Hit Ok to submit new Assignment details
  4. OR hit cancel to exit new assignment dialog without making a new assignment

Modifying an Assignment

  1. Click on ‘Modify Assignment’ to bring up the dialog to modify an assignment
  2. Enter in changed values for the Assignment Name, Weight, Total Score and Category fields
  3. Hit Ok to submit new Assignment details
  4. OR hit cancel to exit modify assignment dialog without making any changes to the assignment

Deleting an Assignment

  1. Click on ‘’Delete Assignment’ to bring up the dialog to delete an assignment
  2. Hit ok to confirm the warning text and delete the assignment
  3. OR hit cancel to exit the delete assignment dialog without deleting the assignment

Adding a New Student

  1. Click on ‘Add New Student’ to bring up the dialog to add a new student
  2. Enter in changed values for the Student Name, EMPL ID, Nickname, Group, Email and Phone fields
  3. Hit Ok to submit new Student details
  4. OR hit cancel to exit new student dialog without making a new student

Modifying a Student

  1. Click on ‘Modify Student’ to bring up the dialog to modify a student

  5. OR hit cancel to exit modify student dialog without making any changes

Adding a New Category

  1. Future implementation: Click ‘Assignment’ in the menu bar and then ‘New Assignment Category...’
  2. Current Implementation: Click on the link for addCategory.html
  3. Enter values for the parent category, category name, and weight
  4. Hit ‘Add Category’ to submit new Category details

Modifying a Category

  1. Future Implementation: Right click the category to edit and click ‘Edit Category...’
  2. Current Implementation: Click on the link for editCategory.html
  3. Enter new values for the parent category, category name and weight
  4. Hit ‘Edit Category’ to submit new Category details

Deleting a Category

  1. Future Implementation: Right click the category to delete and click ‘Delete Category...’
  2. Current Implementation: Click on the link for deleteCategory.html
  3. Click ‘Yes’ to delete the category, and ‘No’ the cancel the action and close the dialog
