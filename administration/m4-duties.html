Mike Ryu

Add a New Class

  1. Open Google Chrome
  2. Go  to http://app.nodegrader.com/ and log in with:
  1. ID: admin
  2. PW: admin
  1. Go to http://app.nodegrader.com/addNewClass 
  2. Click the ‘Add New Class’ button to have the editable dialog box appear.
  3. Fill out the class information (Class Code and Number)
  4. Click ‘OK’ button to add a new class.
  5. Functionality for modifying the course list on the server is temporarily disabled.

Server-side data validation have been implemented, but have not been tested.

If all or part of the class information is missing, an alert pop up show up to instruct the user to enter all information.

Modify an Existing Class

  1. Open Google Chrome
  2. Go  to http://app.nodegrader.com/ and log in with:
  1. ID: admin
  2. PW: admin
  1. Go to http://app.nodegrader.com/modifyClass 
  2. Click ‘Modify Class Information’. This is a mockup for a trigger that a right-click event on a class title will trigger. An editable dialog box with current class information filled-in will appear.
  3. Make the desired changes to the class information.
  4. Click ‘OK’ button to apply the changes.
  5. Navigate back to http://app.nodegrader.com/. The first class listed on this page now should reflect the changes you made using the dialog

Server-side data validation have been implemented, but have not been tested.

If all or part of the class information is missing, an alert pop up show up to instruct the user to enter all information.er all information.

Add an Assignment

  1. Open Google Chrome
  2. Go to http://app.nodegrader.com and log in with:
  1. ID: jsnow
  2. PW: jsnow
  1. Go to http://app.nodegrader.com/newAssignment 
  2. Click the ‘Add New Assignment’ button for editable dialog boxes to appear
  3. Change the values for assignment information in each text box (all fields are required)
  4. Click the ‘OK’ button to add the assignment

A dialog box should pop up upon successfully adding the assignment

Modify an Assignment

  1. Open Google Chrome
  2. Go to http://app.nodegrader.com and log in with:
  1. ID: jsnow
  2. PW: jsnow
  1. Go to http://app.nodegrader.com/modifyAssignment Click the ‘Modify Assignment’ button for editable dialog boxes to appear
  2. Change the values for assignment information in each text box (all fields are required)
  3. Click the ‘OK’ button to add the assignment

Refresh the page and click the ‘Modify Assignment’ button again to see that your changes to the assignment have persisted in the database.

Delete an Assignment

  1. Open Google Chrome
  2. Go to http://nodegrader.com/implementation/public/templates/deleteAssignmentView.html
  3. Click the ‘Delete Assignment’ button
  4. Click the ‘OK’ button to delete the assignment or the ‘Cancel’ button to close the dialog

A dialog box should pop up upon successfully deleting the assignment

Grant Campanelli

Pie Chart / Histogram

  1. Open Google Chrome
  2. Go to app.nodegraders.com/charts 
  3. Change the values for the minimum grades possible on the page.
  4. Click the ‘Update’ button to change the minimum grades

The histogram chart will update its color for grades that are now within those grade ranges. The pie chart data areas will expand and contract based on the number of grade in the updated grade ranges.

Grant Plaster

Add a Student

  1. Open Google Chrome
  2. Go to http://nodegrader.com/implementation/public/templates/addStudentView.html
  3. Click the ‘Add New Student’ button for editable dialog boxes to appear
  4. Change the values for student information in each text box (Student name and EMPL ID are required)
  5. Click the ‘OK’ button to add the student

A dialog box should pop up upon successfully adding the student, and the student name and EMPL ID will be output back out as an alert for confirmation.

Modify a Student

  1. Open Google Chrome
  2. Go to app.nodegrader.com and log in with:
  1. ID: admin
  2. PW: admin
  1. Click the ‘Modify Student’ button for editable dialog boxes to appear
  2. Change the values for student information in each text box (right now, data is canned and already inputted as though the user had previously selected a student to modify)
  3. Click the ‘OK’ button to confirm modifications

You will be returned to the original screen and can see upon further modifications that all data is now persisting and being pulled from the database.

Matt Bleifer

Modify a category

  1. Open Google Chrome
  2. Go to http://app.nodegrader.com and make sure you are logged in
  3. Go to http://app.nodegrader.com/modifyCategory
  4. Choose a category to edit from the dropdown menu
  5. Change any desired fields in the Edit Category dialog
  6. Click the ‘Ok’ button to submit the edited category or the ‘Cancel’ button to close the dialog

You will be returned to the original screen and can see upon selecting more categories that all data is now persisting and being pulled from the database
