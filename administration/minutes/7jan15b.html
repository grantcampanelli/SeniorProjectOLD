Client Meeting: Grader Project

Discussion Topics and Corresponding Details

  • Basic Idea

    • Like Excel, but specialized for grading without any extra features and notations
      • Has built-in spreadsheet engine
      • Include columns such as: Name, UID, Empid…
      • Spreadsheet tallys total score and determines letter grade
      • No built-in names; user-customizable column headings
      • “Add Item” feature for adding graded assignment or category
        • A category has
          • Name (e.g. homework)
          • Parent (optional for making sub-categories)
          • Must support limitless hierarchy
          • Raw Score (optional)
          • Weight (%; required)
        • Late policy
          • E.g. Dr. Mammen’s)
          • Grace day policy
        • Some items may be greyed out in “Add item” dialogue depending on the context of category creation
        • When OK button is pressed, Column headings are generated in the main table
          • Each category will have sub-column headings
          • For example, 6 different homeworks, 5 programs … etc.
        • Column headings needs to be expandable/collapsable
          • Collapsed column will aggregate all sub-column headings
      • Hierarchy of class members
        • Teams and subteams
          • Similar idea as hierarchy of categories
        • Maybe hierarchy is not necessary, just a way to account for sub-groups within the class will be sufficient
      • Statistics
        • Number of submissions
        • Mean/Median/StdDev for each category and sub-column headers
        • Similar to Excel’s functionality
        • This should be lined up with category column headers

  • Graphical Interface

    • “Histogram of Total”
    • With text: Row for every score 0-100; each row contains stars (representing numbers of students)
    • Should accept an input for [letterGradeBorderLine] as a mouse drag input to change the class curve.
    • Pi-Chart interface will be equivalent and acceptable solution
    • Student’s View
      • View as much as instructor except for other student’s information
      • “You are here” feature
      • “What do I need to get it?” feature
        • Based on desired grade
        • Based on future grade
        • This should also be available to the instructor

  • Example Software

    • PeopleSoft
    • Excel
    • PolyLearn
